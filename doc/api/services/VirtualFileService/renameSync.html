Synchronously renames this file system entity

Returns a IFilesystemEntityService instance for the renamed entity. If newPath identifies an existing entity of the same type, that entity is removed first.

Implementation

@override
Result<IFilesystemEntityService, IOError> renameSync(String newPath) {
  return switch (_io) {
    Some() => _io.unwrap().move(path, newPath),
    None() => Err(IOError.doesNotExist),
  };
}
